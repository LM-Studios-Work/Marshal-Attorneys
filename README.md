# Marshal Attorneys

A modern legal services website built with Next.js and TypeScript.

## Features

- Responsive design
- TypeScript support
- Component-based architecture
- Tailwind CSS styling

## Getting Started

### Prerequisites
- Node.js (v18+)
- pnpm

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
pnpm build
pnpm start
```

## Project Structure

- `/app` - Next.js app directory
- `/components` - Reusable UI components
- `/lib` - Utility functions and helpers
- `/hooks` - Custom React hooks
- `/public` - Static assets
- `/styles` - Global styles
